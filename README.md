# 🌿 REVIS

**Revis** — учебный проект на C, упрощённая реализация системы контроля версий (как Git, но для души и учёбы).



## ✨ Возможности (на данный момент)

- ✅ `init` — инициализация репозитория  



## 📌 Текущий статус

> В данный момент активно дорабатывается функция `commit()` — создает коммит и записывает его хеш в HEAD.

Проект находится в активной разработке, но уже можно:
- создавать репозиторий,
- сохранять файлы как blob-объекты,
- вычислять хеши содержимого (SHA-256).



## 🗂️ Структура проекта
```
revis/
└── core/
  ├── src/ — реализация функций (.c)
  ├── include/ — заголовочные файлы (.h)
  └── src/main.c — точка входа
```



## 🛠️ Сборка

> Зависимости (установка Ubuntu/Debian):
```bash
sudo apt install openssl build-essential zlib1g-dev
```

```bash
git clone https://github.com/tracegillonsign/revis.git
cd revis/core
make
```
> После сборки исполняемый файл **revis** появиться в core/build/


## 📖 Примечание

Проект не завершён и создан в учебных целях — для изучения:
- системного программирования на C
- работы с файловыми структурами
- управления памятью и указателями
- основ работы систем контроля версий
