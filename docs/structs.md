### `struct Objects`

Представляет собой элемент дерева. Файл либо директорию.

- `char type[5]` — тип: `"blob"` или `"tree"`
- `char *name` — относительный путь к файлу. Выделяется через `strdup()`
- `char hash[65]` — хеш содержимого. Оканчивается на `\0`

**Память:**
	После использования освободить `name` для каждого объекта внутри структуры. Затем освободить сам массив `struct Object*`.
